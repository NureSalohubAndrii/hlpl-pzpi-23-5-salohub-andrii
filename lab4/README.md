# Лабораторна робота №4

### Рівень 1

Розробіть базу даних для соціальної мережі з таблицями: Користувачі, Друзі, Пости, Коментарі, Лайки. Визначте зв'язки та обмеження для забезпечення цілісності даних.

### Рівень 2

Створіть моделі бази даних соціальної мережі. Реалізуйте додавання нових постів та коментарів.

### Рівень 3

Впровадьте функціонал пошуку та фільтрації постів у соціальній мережі.

### Рівень 4

Розробіть систему логування та моніторингу запитів у соціальній мережі. Зберігайте логи у базі даних та надайте інтерфейс для їх перегляду адміністраторами.
